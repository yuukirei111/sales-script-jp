## ⭐ 黄金 SNS 化原则(2026-04-28 重大修正)

**之前误区**:塞间投词(ねえ / なんか / てか)、犹豫句(〜的な / 〜って感じ?)、内心 OS(○○って思った)以为是"SNS 化"。**❌ 错。爆款里这些一个都没有。**

**真正的 SNS 化** = 直球播报 + 对的关键词。具体共性如下:

### 1. 钩子复用 = 安全
3 条爆款的钩子都来自 formulas.md 既有公式句,**强力复用**,不臆造:
- `これ買おうとしてる人、ちょっと待って！` (出现 2/3)
- `もう隠さなくていいよ。` (出现 1/3)
- `アンニョン!見て、このXX爆誕。` (出现 1/3)

### 2. 必须有"实证元素"≥ 2 项
- **具体成分名**:`ナイアシンアミド` `PDRN` (不是泛指"浓厚成分")
- **量化人设**:`毎年300種類のスキンケアを試して`
- **具体场景**:`塗って寝た翌朝の』
- **具体身体感**:`パンッとしたハリとツヤ感』
- **第三方事实**:`Amazonでも即完売した` `バズりすぎて買えなかった』

### 3. SNS 高频热词(必带 ≥ 1)
- `爆誕`(可重复用作节奏点:`ツヤ肌爆誕』→`ハリ爆誕』)
- `幻のアイテム』
- `即完売』
- `バズりすぎて』
- `〜ちゃう`(`買えちゃう』 — 轻松感)
- `救世主』
- `糸引く濃厚』(产品质地视觉化)
- `肌の奥から〜』
- `レベチ』

### 4. CTA 黄金模板(几乎不换)
**3/3 都用**:`こっそりチェックしてみてね』 / `こっそりチェックしてね』

可微调:
- `(在庫なくなる前に、)こっそりチェックしてみてね』
- `(今ならAmazonの半額以下で買えちゃうから、)こっそりチェックしてね』

❌ **严禁**命令式动词当主动作:`急いで』`カートへ』`チェックして!』(去掉 `〜てね/〜てみて』软化)
❌ **严禁**"邀请式过度软化":`合うかわかんないけどさ』`覗いてみ?』(过头变 ChatGPT 风)

### 5. 节奏 = 极简直球
- Script 3 极短,7 句话搞定一条 — 信息密度爆表
- 不留废话、不留犹豫、不解释自己为什么"也想买"
- "我在跟你播报新发现",**不是**"我在跟你聊天"

## 新发现的角度(formulas.md 10 子类之外)

### 【観察期】教育.反直覚損失(Reverse Loss Education)— 2026-05-18 落盘(#8 由来)

**核心ロジック**
受象が「正しい」と信じている使い方/行動を否定し、「あなた、実は損してる」と通告する否定型ティーチ。「これから損する」FOMO ではなく「**すでに損している**」現在進行形損失で刺す。

**典型句**
- `でも、{受象細分行動}してる人、めっちゃ損してる』(#8 5/11 5 単で実証)
- `{受象}の人、これ間違えてるかも』(候補)
- `{一般的な使い方}してたら、{商品}の本気が出ない』(候補)

**適用条件**
- 商品に「**正しい使い方 vs 一般的な誤用**」差分が存在する(例:VC = 朝も夜も、UV = 曇りの日も、レチノール = 段階的に等)
- 受象の既存ユーザー層 / 部分認知層を狙撃可能(初心者向けではない、既に商品/カテゴリを知っている層が対象)
- 「教育 → 解答 → 価格 → 死線」漏斗構造を支える商品ストーリーがある

**禁止 / リスク**
- ❌ 説教調にならない(梨花調の生口語で軽くやる:`めっちゃ損してる』≪ 学術調)
- ❌ 否定だけして解答を出さない(必ず「正解は◯◯」で続ける)
- ❌ ターゲットが不明瞭(`使ってる人』だけだと受象を絞れない、`夜だけ使ってる人』のように細分必須)

**観察期 → 子類昇格基準**
連続 2 回出現 + いずれも >3 単爆款 → L2 提案で formulas.md「教育型」の正式子類として新設(用户確認待ち)。

- 用户指出后承认:**日语里没有 `肌に空気入った』 这种成立的惯用表达**,字面读会让人困惑(空气从哪进的?进哪了?)
- 错误根源:**为了显得"有创意"硬造一个不存在的比喻** + 用"观众自己脑补"为词不达意辩护
- 替换方案:`しぼんでた肌がぷくっと戻ってきた』(`しぼむ→ぷくっと』 是日语里日常用来形容气球 / 果实 / 皮肤的真实反义词组合)
- **教训沉淀为 R8 规则**(SKILL.md 规则 16,CRITICAL 级):严禁自创日语比喻,只能用 winning-patterns / formulas 已验证句或日语真实存在的惯用表达

### 词语境错配 `ジャーずしっと重い』 — R8 增补反例(2026-05-06)
