## 全局规则(Global Logic)

1. **语言**:100% 纯 Tame-guchi(平语 / 闺蜜语)。**严禁 です/ます**
2. **断句美学**:每句 8–12 字。多用「。」「!」和换行,模拟 TikTok 0.85–1.1 倍速口播节奏
3. **情绪词**:必带 `マジで` `ガチで` `えぐい` `ヤバい`
4. **拟声词**(灵魂):按肤感插入 `パンッ!` `うるうる` `ツヤツヤ` `ヒリヒリ` `すーっと` `もっちり`
5. **结构骨架**:`[视觉阻断钩子] → [核心诉求展开] → [物理/背书说服] → [偷感 CTA]`
6. **价格语气**:震惊 + 私密并存,像发现系统 Bug 或偷偷分享内部福利
7. **特定语气助词**:`～だよ` `～じゃん` `～っしょ`

## 一、效果诉求型 (Effect-Oriented)
> 核心逻辑:建立 [现状] 与 [未来] 的巨大落差。

### A. 负面刺痛型 (Pain Resolution)
**针对**:30–50 代,有深层焦虑、追求"解决问题"的刚需人群。

| 时间段 | 阶段 | 逻辑 | 金句库 |
|---|---|---|---|
| 0–3s | Hook 拦截 | 否定现状 + 危机拦截 | `もう隠さなくていいよ。` / `これ買おうとしてる人、ちょっと待って!` / `鏡見るのが嫌だった人、集合。` |
| 3–8s | Pain 刺痛 | 揭开伤疤 + IP 深度共情 | `コンプレックスの塊だった私を救ってくれたのがこれ。` / `老け見えの原因、実はこれだったんだよ。` / `鏡見るのが嫌だったあの頃の私に教えたい。` |
| 8–13s | Rescue 救赎 | 物理质感展示 + 权威填补 | `この糸引く濃厚PDRN、見て?溝を埋めるみたいにパンッ!だよ。` / `シワ・たるみの救世主って言われる理由、使えばわかる。` / `肌の奥からパンッ!だよ。` |

### B. 正面憧憬型 (Aspiration)
**针对**:18–35 代,追求精致感、韩流审美、高 QOL 的人群。

| 时间段 | 阶段 | 逻辑 | 金句库 |

### C. 反直覚損失型 (Reverse Loss Education) — 2026-05-18 落盘(#8 由来、観察期 1/1、用户 ✅)

**针对**:已使用某品类但**用法存在常见误用**的中阶用户。**初心者不適用**。

| 时间段 | 阶段 | 逻辑 | 金句库 |
|---|---|---|---|
| 0–3s | Hook 拦截 | 阻断 + 既存知識への揺さぶり | `これ買おうとしてる人、ちょっと待って!』 / `{品类}使ってる人、これ間違えてるかも』 |
| 3–8s | 信任锚 + 価格信号 | L1-L4 信任锚 + R10 模糊価格 | `話題の{愛称}、今{月}セールで半額近く』 |
| 8–13s | ★ 反直覚損失 ★ | 受象細分 + 否定型ティーチ + 損失 framing | `でも、{特定使用方式}してる人、めっちゃ損してる』 / `{受象}の人、これ間違えてるかも』 |
| 13–18s | 教育解答 | 原因 → 行動指針(正解提示) | `{成分/メカニズム}、{効果}に効くから、{正解行動}するのが正解』 |
| 18–22s | 死線 | R20 具体日付推奨 | `セールは{日付}まで』 |
| 22–25s | CTA | R11 既定款 | `左下からチェックしてみてね』 |

**適用条件**:商品に「正しい使い方 vs 一般的な誤用」差分存在 / 既存ユーザー or 部分認知層狙撃(初心者ターゲット NG) / 6 行極簡推奨 / 漏斗式 framing 必須(R29 例外条款適用)

**禁止 / リスク**:
- ❌ 学術調 / 説教調(梨花調の生口語必須:`めっちゃ損してる』 ≪ 学術的説明)

- ❌ 受象が不明瞭(`使ってる人』 だけだと弱い、`夜だけ使ってる人』 のように細分必須)

**現状検証**:1/1(#8 のみ、観察期。連続 2 回出現 + いずれも >3 単 → 正式子類昇格基準達成)

**実証案例**:`winning-patterns.md` #8 梨花 ビタテクト 5/11 5 単爆款

## 二、价格诉求型 (Price-Oriented)
> 核心逻辑:利用 [外部权威锚点] 制造 [由于系统错误导致的低价] 错觉。
>
> 共通拟声词:`バタバタ` `ドキドキ` `ガバっと`(烘托抢购氛围)

### A. 平台降维比价型 (Platform Anchor)
**借用 Amazon / 乐天的价格作为锚点。**

| 阶段 | 金句库 |
|---|---|
| Hook - 利益核弹 | `Amazon半額以下!!` + `マジでバグってるよ、これ。` |
| Evidence - 事实对比 | `普通にAmazonで買ったら3,500円。` + `でも、ここなら2,000円で買える。` |
| Reason - 合理化 | `TikTokショップの大感謝祭、本気出しすぎ。` |
| CTA - 偷感转化 | `損したくない人、今すぐ左下のカート見て。` |

### B. 专业博主"捡漏"型 (Expert's Bargain)
**用"专业人士都惊了"对冲廉价感。**

| 阶段 | 金句库 |
|---|---|
| Hook - 阻断 | `正直、教えたくないんだけど。` + `この値段はえぐい。` |
| Authority - 人设 | `毎年300種類のスキンケア試す私が、ガチでストック買いした。` |
| Logic - 高价值感 | `成分(PDRN)もレベチなのに、この値段は事件だよ。` |
| CTA - 行动 | `在庫なくなる前に、今のうちに確保して!` |

### C. 隐秘福利 / 特权型 (Secret Privilege)
**包装成"只有刷到视频的人"才有的专属特权。**

★ **2026-05-23 真戦実戦実証済**(#10 マリナ ハリアス 5/18 真出单 5 単 / CTR 4.25%)

| 阶段 | 金句库 |
|---|---|
| Hook - 拦截 | `これ買おうとしてる人、ちょっと待って!` + `限定クーポン、見つけた。` |
| Privilege - 特权感 | `普通に検索しても出てこない。この動画からしか手に入らない。` |
| Urgency - 消失恐惧 | `枚数限定。消えちゃう前に使わないと損。` |
| CTA - 指令 | `こっそり下のリンクからクーポン取って。` |

#### 正式公式テンプレ(#10 5/18 真出单 5 単 + #5 用户修正版、骨架 95% 共通)

```
[winning hook + 福利钩]    これ買おうとしてる人 ちょっと待って 限定クーポン 見つけちゃった
[隠秘性圧縮 + 商品]        普通に検索しても出てこない この動画からしか取れないやつ この{愛称}
[L3信任锚 + アンカー対比]  {カテゴリ内地位言及} 公式{価格}のやつ 今ならTikTok限定で{X%}オフ
[D型 scarcity + CTA]       枚数限定だから 左下のカートからチェックしてみて
```

**コア核 — 稀缺密度三層押し**:
1. **時間/数量曖昧**:`限定クーポン』
2. **空間限定**:`この動画からしか取れない』
3. **数量限定**:`枚数限定』

3 件並列で説明なしに FOMO 直撃 → 説得不要、極限圧縮可(4 行下限、SKILL.md R21 補足 2)。

**適用条件**
- アンカー価格(公式 / Amazon 等)が言える
- 「クッション界の王様」「コスメ界の救世主」等の L3 口碑表現が成立する商品
- 説得 framing(成分名 / 効果)不要、純粋 FOMO 駆動
- 駆動エンジン = **CTR(钩子拉力)**(#10:4.25%)

**禁止 / リスク**
- 説得 framing を混ぜると 4 行に収まらず、隠秘福利の純度が下がる → R29 単一 framing 必須
- 「Amazon ◯◯円のところ TikTok △△円!」と数字 2 つ並べると R3 折扣去重違反
- CTOR 弱め(#10:1.37%)= 進橱窗后の説得力は他 framing に劣る → 流量基数が必要

**L3 信任锚表現バリエーション**:
- `{カテゴリ}界の王様』(#10 実証)
- `バズりすぎて買えなかった幻のアイテム』(#2 用户喂入)
- `Amazonでも即完売した』(#1 用户喂入)
- `ずっと品切れ続きだったやつ』(L3 候補)
- `製薬会社共同開発の本気設計』(L2-L3 間、商品が薬用系の時)

### D. 节点大促截流型 (Event Hijack)
**绑定国民节点(GW、新生活)给冲动消费一个正当理由。**

| 阶段 | 金句库 |
|---|---|
| Hook - 紧急状态 | `GWセール、最終日!!` + `ヤバい、もう全部売り切れる。` |
| Offer - 极限优惠 | `年に一度の神セール。今だけAmazonの半額以下。` |
| Warning - 过期无效 | `明日には3,500円に戻っちゃうよ。` |
| CTA - 冲刺 | `販売終了しちゃう前に、下のカゴへ急いで!` |

## 三、在库告急型 (Urgency-Oriented)
> 核心驱动:**FOMO Engine**(损失规避 + 从众压力 + 合理稀缺)
>
> 共通节奏:语速极快,「。」「!」频繁。情绪词 `ギリギリ` `パニック` `争奪戦`。
>
> 黄金四段位:`[Hook 紧急通知] → [Evidence 缺货证据] → [Threat 损失警告] → [Action 购物车指令]`

### A. 再入荷型 (Restock Miracle)
**针对观望者 + 之前没买到的人。**

| 阶段 | 逻辑 | 必选金句 |
|---|---|---|
| Hook | 惊喜大喊 | `やっと再入荷したよ!` |
| Evidence | 揭露断货历史 | `正直、次いつ入るか不明💦` / `Amazonでもずっと品切れ中💦` |
| Scarcity | 极少量到货 | `幻のサーモンクリームが奇跡的に再入荷。` / `幻のアイテム` |
| CTA | 警告再次断货 | `販売終了しちゃう前に、今すぐ下のカゴへ急いで!` |

### B. 终了间际型 (Final Countdown)
**活动结束前 24h 发布,主打死线。**

| 阶段 | 逻辑 | 必选金句 |
|---|---|---|
| Hook | 时间死线 | `今夜23:59で、この価格終わっちゃう。` / `緊急事態!!もうチェックした?` |
| Evidence | 价格回升预告 | `明日には3,500円に戻るから注意して!` |
| Threat | 损失计算 | `これが最後のチャンス。明日にはもう元値に戻っちゃうよ。` |
| CTA | 强制下单 | `後悔してほしくないから動画撮ってる。今すぐカートへ。` |

### C. 枚数限定型 (Limited Coupon)
**针对价格敏感人群,主打抢券。**

| 阶段 | 逻辑 | 必选金句 |
|---|---|---|
| Hook | 特权公开 | `限定クーポン、あと10枚しかなかった…` |
| Evidence | 实时剩余感 | (动态报数,如 `さっき見たら残り5枚`) |
| Scarcity | 抢夺竞争 | `早い者勝ちだよ。` |
| CTA | 领取指令 | `今この動画を見た人だけ、ギリギリ間に合う。` |

### D. 他店完售型 (Cross-Platform Scarcity)
**针对理性比价人群,主打全网稀缺。**

| 阶段 | 逻辑 | 必选金句 |
|---|---|---|
| Hook | 大平台售罄截图 | `Amazonも楽天も即完売したこれ。` |
| Evidence | 本渠道独家余货 | `ここだけ、奇跡的に在庫残ってた!` |

| CTA | 行动指令 | `この動画見たあなたが最後の一人かも。` |

## 四点六五、SNS 自然口播连接模板(2026-04-28 新增,基于 #5 用户修改)

### 时间副词(必带 ≥ 1)
- `今なら〜』(全部 4/4 爆款都用)
- `今だけ〜』
- `今ならTikTok限定で〜』

### 状态化折扣表达(替代静态价格)
- ✅ `今ならTikTok限定で43%オフになってる』(动词化、自然口播)
- ✅ `今だけ半額以下で買えちゃう』(`〜ちゃう』 轻松感)
- ❌ `TikTok限定で2000円。43%オフ。』(双数据冗余 + 句号硬断)

### 价格表达模糊化(R10,2026-05-06 落盘)

| ✅ 推荐(SNS 闺蜜分享语气) | ❌ 避免(广告播报感) |
|---|---|
| `千円台』 | `1,200円』 |
| `二千円台』 | `2,000円』 |
| `三千円切ってる』 | `2,980円』 |
| `Amazonの半額以下』 | (具体数字) |
| `まさかの千円台』(`まさかの』+ 模糊范围)| `まさかの1200円』 |

**例外保留**:锚点价格对比 / 具体折扣率(`43%オフ』)/ 结构性数字(`3日だけ』)豁免。R10 只针对"绝对单价"表达。

**落盘案例**:用户 2026-05-06 修改 `まさかの1200円』 → `まさかの千円台』,理由:具体数字读起来像广告播报,模糊范围更口语 SNS 化。

**2026-05-14 用户確認**:アンカー対比文脈で `千円台』 と `半額以下』 は **両方併用可**(どちらが正解 ≠)。`公式は3520円なのに...』 直前なら `半額以下』 でショック増、単発時は `千円台』 で SNS 化。状況に応じて選ぶ。

- 反例 → 见 cta-library 反例库(NG-CTA-001 ~ NG-CTA-006)

### Quotable 复合名词模板(替代主谓断句)
- ✅ `30代肌リセット』 / `ツヤ肌爆誕』 / `ハリ爆誕』 / `ツヤ復活』
- ✅ `毛穴ゼロ肌』 / `くすみ抜け肌』
- ❌ `30代肌、リセット。』(顿号断了 quotable 节奏)
- ❌ `ツヤ肌が爆誕します。』(主谓 + 敬语)

### 句末 SNS 化语尾
- `〜になってる』(状态描述)
- `〜ちゃった』 / `〜ちゃう』(降低决策压力)
- `〜の。』(像 #1 `入ってるの。』,亲密感)

### 折扣表达去重决策树
价格信息池:`Amazon锚点 / 半額以下 / X%オフ / 具体价格』
- 选 1:简洁但缺少冲击力(参照 #3:仅 `半額以下』)
- **选 2(推荐)**:`Amazon锚点 + X%オフ』 或 `半額以下 + X%オフ』
- 选 3:啰嗦冗余,降低 SNS 直球感
- 选 4:严禁

## 六、Quality Gate(自检清单)

### 0. R30 本地化优先(CRITICAL — 凌驾所有其他项,2026-05-07 落盘)

每条脚本写完,内心朗读一遍,问:
1. 这句日语,日本人会真的这样说吗?
2. 这条脚本读起来像「日本人写的」还是「翻译来的」?
3. 有没有为了凑某个规则而加的句子?

任一回答勉强 → 删 / 改。**R30 不通过,后面的 Quality Gate 不用看了 — 直接重写。**
